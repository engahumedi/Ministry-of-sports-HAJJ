خطوات ربط قاعدة البيانات
مجانية — تكفي لآلاف السجلات — آمنة بالكامل
١ أنشئ مشروعاً في Firebase
افتح console.firebase.google.com ← Add project ← اختر اسماً مثل hajj-maisam-1447 ← أكمل الإعداد.
٢ فعّل Firestore Database
من القائمة الجانبية ← Firestore Database ← Create database ← Start in production mode ← اختر region قريبة ← Enable.
٣ فعّل المصادقة Authentication
من القائمة ← Authentication ← Get started ← اختر Email/Password ← فعّله ← Save.

ثم من تبويب Users ← Add user ← أدخل إيميلك وكلمة مرور قوية. هذا هو حساب المشرف الوحيد الذي يقدر يفتح لوحة البيانات.
٤ اضبط قواعد الأمان
من Firestore ← Rules ← استبدل النص الموجود بما يلي ← Publish:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /hajj_attendance_1447/{doc} { // أي شخص يقدر يسجل حضوره — مع التحقق من صحة البيانات allow create: if request.resource.data.keys() .hasAll(['name','id','time','date']) && request.resource.data.name is string && request.resource.data.name.size() >= 4 && request.resource.data.id.matches('[12][0-9]{9}'); // القراءة والحذف للمشرف المسجّل دخوله فقط allow read, delete: if request.auth != null; // لا أحد يقدر يعدّل سجلاً بعد إنشائه allow update: if false; } } }
